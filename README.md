# 成衣購物網
https://asianiceguy9423.github.io/clothesshop/
<hr>

### 使用技術
- 使用Vue3框架完成。
- 使用bootstrap搭配製作響應式網站。
- 使用暫存來記錄購物車商品。
- 彈跳視窗提示使用者送出表單是否正確。
